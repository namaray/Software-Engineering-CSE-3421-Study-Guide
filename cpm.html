Critical Path Method (CPM)

Course: CSE 3421 | Topic: Project Scheduling

1. Basic Definitions

Critical Path Method (CPM): A method used in project planning to determine the earliest time by which the whole project can be completed.

  • Critical Task: A task/activity which cannot be delayed. If it is delayed, the whole project is delayed.
  • Critical Path: The sequence of critical tasks. It is the longest path through the network but represents the minimum time required to complete the project.
  • Float (Slack): The amount of time a task can be delayed without delaying the project.
    Formula: Float = LS - ES or LF - EF.

2. The Problem Statement

We need to find the Critical Path and Duration for the following project activities:

Activity Duration (Weeks) Precedents (Depends on)
A6- (Start)
B4- (Start)
C3A
D4B
E3B
F10- (Start)
G3E, F
H2C, D

Node Legend (Activity-on-Node)

Top Row (Blue): Forward Pass (Earliest Start / Earliest Finish)
Middle Row: Activity Name
Bottom Row (Orange): Backward Pass (Latest Start / Latest Finish) & Float

3. Step-by-Step Solution

Step A: Forward Pass (Earliest Times)

ES = Max(Predecessor EF) EF = ES + Duration

Start from Left (0) → Move Right.

  • Start (A, B, F): Start at 0.
  • C: After A(6). Starts 6. Finishes 6+3=9.
  • D: After B(4). Starts 4. Finishes 4+4=8.
  • E: After B(4). Starts 4. Finishes 4+3=7.
  • G (Merge): After E(7) AND F(10). Take MAX(7, 10) = 10. Finishes 10+3=13.
  • H (Merge): After C(9) AND D(8). Take MAX(9, 8) = 9. Finishes 9+2=11.

Project Duration: The FINISH node depends on G(13) and H(11). Max is 13 Weeks.

Step B: Backward Pass (Latest Times)

LF = Min(Successor LS) LS = LF - Duration

Start from Right (13) → Move Left.

  • End: Set LF for last nodes (G, H) to 13.
  • G: LF=13. LS = 13-3 = 10.
  • H: LF=13. LS = 13-2 = 11.
  • F: Before G(10). LF=10. LS=0.
  • C: Before H(11). LF=11. LS=8.
  • D: Before H(11). LF=11. LS=7.
  • E: Before G(10). LF=10. LS=7.
  • A: Before C(8). LF=8. LS=2.
  • B (Burst): Before D(7) AND E(7). Take MIN(7, 7) = 7. LF=7. LS=3.

4. Complete Network Diagram

Final Analysis:
  • The activities with Float = 0 are Start → F → G → Finish.
  • This is the Critical Path.
  • Total Duration: 13 Weeks.
